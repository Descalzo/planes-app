# Planes Frontend

Proyecto React + Vite para el frontend del MVP de planes y chat.

## Estructura inicial

- `src/pages/`: páginas de la aplicación
- `src/components/`: componentes reutilizables
- `src/services/`: API cliente y servicios
- `src/routes/`: rutas de la aplicación

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
